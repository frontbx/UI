Navbar

Navbar provides a foundation for building out various navigation components.

  • Markup
  • Links
  • Logo

Markup

The base .navbar component sets the core foundation for building a navigation component. Navbar can then be combined with other FrontBx Components such as .nav-menu, Tabs, Buttons, Dropdown and more.
